🚀 Go Web Template

基于GoFrame框架，参考private-ai-chat架构设计的Web应用模板

📋 架构特点

  • API层: 控制器扁平+全小写命名
  • Internal层: 内部核心支撑层（业务逻辑、模型、配置等）
  • Utils层: 全局工具函数目录
  • 统一响应封装: 参考go-novel的SuccessEncrypt设计

🔌 API接口

POST /api/user/register

用户注册 - 参数: username, password, email

POST /api/user/login

用户登录 - 参数: username, password

GET /api/user/profile

获取用户信息 - 参数: user_id

GET /api/product/list

获取产品列表 - 参数: page, page_size

GET /api/product/detail

获取产品详情 - 参数: id

POST /api/product/create

创建产品 - 参数: name, description, price

🗄️ 数据库配置

  • 数据库: template_chat
  • 主机: localhost:3306
  • 用户名: root
  • 密码: root
  • 表前缀: ls_
